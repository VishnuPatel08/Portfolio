# Portfolio Assets

This folder contains various assets for your portfolio website.

## Contents:

- **resume.pdf** - Your downloadable resume/CV
- **favicon.ico** - Website favicon (16x16, 32x32, 48x48px)
- **icons/** - Custom icons and graphics
- **documents/** - Additional downloadable documents

## Resume Requirements:

Your resume should be:
- Professional and well-formatted
- Up-to-date with current experience
- Optimized for both digital viewing and printing
- Named clearly (e.g., "FirstName_LastName_Resume.pdf")

## Favicon Guidelines:

- Create multiple sizes: 16x16, 32x32, 48x48, 64x64px
- Use simple, recognizable design
- Ensure good contrast for visibility
- Test on different backgrounds

## File Organization:

Keep this folder organized with:
- Clear naming conventions
- Appropriate file formats
- Optimized file sizes
- Version control for updates
